O círculo centralizado

Em primeiro lugar para obtermos um círculo vamos criar um quadrado com a tag <div> e aqui escolhemos atribuir uma pseudo-class intitulada por x0. Após isso basta estilizar a div com cor(background), largura(width) e altura (height), neste caso cor, largura e altura com escolhas a critério de cada um. Um alvo com 3 setores

Agora vamos criar um alvo circular com três setores. Para tal basta inserir 3 divs ou caixas com pseudo classes diferentes. E para estilizar em forma de círculos procedemos como anteriormente. No entanto iremos usar cores e dimensões distintas para cada caixa. Vejam os códigos em style. Após isso veja como fica nosso alvo.

Um anel

Para criarmos um anel iniciamos com uma caixa quadrada que iremos usar inicialmente uma cor distinta da desejada para o anel, por exemplo background-color: black;. Seguindo formaremos a caixa com dimensões width: 100px;, height: 100px; e a centralizamos com margin: auto;. Para finalizar inserimos as bordas sólidas com a cor e largura que desejamos para o anel, por exemplo: border: 50px solid blue; border-radius: 100%; e trocamos a cor black para transparent do background inicial.

Um semi anel

Para montar o semi-anel circular optaremos por usar uma cor background-color: black; para a caixa e por fim a mudaremos para transparent, isso para que possamos inserir as bordas. Assim começa-se com um caixa retangular black onde optamos por dimensões: width: 100px; height: 50px; e a centralizamos com margin: auto; e usando border-top-right-radius: 100px; , border-top-left-radius: 100px; deixar o retângulo como um semi-disco. Agora colocaremos as bordas sólidas da cor que desejamos para o anel com border-left: 50px solid blue;, border-right: 50px solid blue; e border-top: 50px solid blue;. Para finalizar basta trocar o background de black para transparent.
