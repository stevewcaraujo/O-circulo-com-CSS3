Apresentações de algumas habilidades do CSS3 utilizando a forma do Círculo

Antes de fazer qualquer comentário sobre o CSS é importante compreender o significado da sigla CSS que é Cascading Style Sheet, ou Folha de Estilo em Cascatas, quando escrevemos CSS3 estamos dizendo que trata-se da versão 3. É importante frizar que o CSS3 não é uma linguagem de programação e sim uma linguagem de marcação de estilos e sendo assim sua aplicação é utilizada na estilização de componentes escritos em linguagens de marcação. De uma forma bem simples após escrever, por exemplo, o conteúdo com HTML5 utilizamos um marcador de estilos, neste caso o CSS3, para dar: estética, modelo ou elegância ao conteúdo tornando ao usuário uma experiência mais agradável. Trocando em miúdos estética, modelo ou elegância ao conteúdo significa que podemos inserir cor nos mais diversos campos(body, h1, h2, ...,h6, e etc.) estilizar os links, inserir imagens, aúdios, construir formas como: quadrados, retângulos, círculos, espirais e muitas outras formas. Além de muitas formas de efeitos. Não conseguerei mencionar todas as formas de estilos que é possível inserir com está fabulosa ferramenta, mas o fato é que empolgante e atrativo o que é possível fazer com o CSS3. Desta forma, resolvi como meu primeiro site apresentar algumas habilidades que aprendi até agora com o CSS3 utilizando a forma do círculo. Então sem mais vamos ao trabalho e espero poder auxiliar outros com esta singela exibição aqui exposta.

Um anel

Para criarmos um anel iniciamos com uma caixa quadrada que iremos usar inicialmente uma cor distinta da desejada para o anel, por exemplo background-color sendo black. Seguindo formaremos a caixa com dimensões de largura com width de 100px, altura com height de 100px e a centralizamos com margin de parametro auto. Para finalizar inserimos as bordas sólidas com a cor e largura que desejamos para o anel, por exemplo:

  • border: 50px solid blue;

Isso para obtermos um anel com espessura de 50px. E por fim aplicamos o arredondadmento do anel quadrado com:

  • border-radius: 100%;
e trocamos a cor black para transparent do background inicial. Assim obtemos o anel desejado, veja:

Um semi anel

Para montar o semi-anel circular optaremos por usar um background-color de cor black, para a caixa e por fim a mudaremos para transparent, isso para que possamos inserir as bordas. Assim começa-se com um caixa retangular black onde optamos por dimensões de largura width de 100px, altura height de 50px e a centralizamos com margin de parametro auto. Obtendo um retângulo. Agora usaremos:

  • border-top-right-radius de 100px
  • border-top-left-radius de 100px

Assim o retângulo ficará um semi-disco. Agora colocaremos as bordas sólidas da cor que desejamos para o anel com:

  • border-left: 50px solid blue
  • border-right: 50px solid blue
  • border-top: 50px solid blue

Algumas das habilidades na construção da espiral que iremos utilizar são:

  • shorthand com bordas como por exemplo border: 20px solid blue(estes parametros são apenas para exemplificar);
